자바스크립트 변수 : typeof 키워드로 자료형 알아보기

형식) typeof 데이터값(변수)

변수 선언 형식) var 변수명 = 저장할 값;

변수의 자료형(예: int, double, ...)은 붙이지 않는다.

템플릿 문자열 = 백틱

1) 여러 데이터형의 결합할 때 +(연결연산자)를 이용하는 대신 문자열 사이에 변수 or 수식을 ${}에 넣어 사용하는 방법이다.

2) 작은 따옴표가 아닌 역쿼트(``)를 사용하여 묶어서 지정한다.

3) 결과형은 문자열형으로 저장(출력)된다.

4) 형식) `〈HTML 태그〉문자열 ${수식 or 변수} 문자열 〈/HTML 태그〉`
